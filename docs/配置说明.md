# 配置说明

配置文件位于 `backend/config.json`。首次运行前需从模板复制：

```bash
cp backend/config.example.json backend/config.json
```

> ⚠️ `config.json` 包含 API Key 等敏感信息，已加入 `.gitignore`，**不会被提交到 Git**。仓库中只有 `config.example.json` 模板。

## 配置项概览

| 类别 | 说明 | 详细文档 |
|------|------|----------|
| 大模型 (LLM) | models、active_model、temperature、max_tokens、think_mode | [API 密钥与模型](./API密钥与模型.md) |
| 语音识别 (STT) | stt_provider、whisper 模型、豆包 ASR、通用 ASR 配置 | [豆包语音识别](./豆包语音识别.md)、下方简要说明 |
| 学习相关 | position、language | — |
| 语音检测 (VAD) | auto_detect、silence_threshold、silence_duration | — |

## 完整字段说明

```jsonc
{
  // ── 大模型配置 ────────────────────────────────────
  "models": [ { "name", "api_base_url", "api_key", "model", "supports_think", "supports_vision" } ],
  "active_model": 0,
  "temperature": 0.7,
  "max_tokens": 4096,
  "think_mode": false,

  // ── 语音识别 ──────────────────────────────────────
  "stt_provider": "whisper",        // "whisper" | "doubao" | "generic"
  "whisper_model": "base",          // tiny/base/small/medium/large-v3
  "whisper_language": "zh",
  // 豆包 ASR（仅当 stt_provider 为 doubao 时使用，见 docs/豆包语音识别.md）
  "doubao_stt_app_id": "",
  "doubao_stt_access_token": "",
  "doubao_stt_resource_id": "volc.seedasr.sauc.duration",
  "doubao_stt_boosting_table_id": "",
  // 通用 ASR（仅当 stt_provider 为 generic 时使用）
  "generic_stt_api_base_url": "https://api.example.com/v1",
  "generic_stt_api_key": "",
  "generic_stt_model": "whisper-1",

  // ── 学习配置 ──────────────────────────────────────
  "position": "后端开发",
  "language": "Python",

  // ── 语音检测 (VAD) ───────────────────────────────
  "auto_detect": true,
  "silence_threshold": 0.01,
  "silence_duration": 2.0
}
```

- **语音识别**：界面设置中可切换「Whisper（本地）」「豆包（API）」或「通用 ASR」；通用 ASR 需要填写兼容 OpenAI `/audio/transcriptions` 的 Base URL、API Key 和模型名。
- **大模型**：各厂商 API Key 申请与示例见 [API 密钥与模型](./API密钥与模型.md)。
