# 音频配置

本助手可捕获 **系统音频**（如在线会议中对方的声音）或 **麦克风**。不同系统配置方式不同。

---

## macOS — BlackHole 虚拟音频设备

macOS 不允许直接录制系统输出，需要借助 **BlackHole** 虚拟音频设备。

### 1. 安装 BlackHole

```bash
brew install blackhole-2ch
```

若无 Homebrew，先安装：[https://brew.sh](https://brew.sh)

### 2. 创建多输出设备

1. 打开 **音频 MIDI 设置**（Spotlight 搜索 "Audio MIDI Setup"）。
2. 左下角 **+** → **创建多输出设备**。
3. 右侧勾选：✅ 你的扬声器/耳机 + ✅ **BlackHole 2ch**。
4. 将 **主设备** 设为扬声器，勾选 BlackHole 的「漂移校正」。

### 3. 设置系统输出

系统设置 → 声音 → 输出 → 选择刚创建的 **多输出设备**。

### 4. 在工具中选择设备

在助手的音频设备下拉框中选择 **BlackHole 2ch**（设备列表中带 ⟳ 标记、归类在「系统音频」下）。

> 使用结束后，回到 系统设置 → 声音 → 输出 改回原来的扬声器即可。  
> 详细图文：[macOS 使用 BlackHole 录制系统声音](https://zhuanlan.zhihu.com/p/667430079)

## Windows — 系统音频采集

### 方式一（推荐）：soundcard WASAPI

安装 `soundcard` 后，设备列表会出现带 **★ (系统音频)** 标记的 WASAPI 设备，音质更好，无需立体声混音：

```bash
pip install soundcard
```

启动后在设备下拉框中选择带 ★ 标记的设备即可。

### 方式二：立体声混音 (Stereo Mix)

若无法使用 `soundcard`，可启用系统自带的「立体声混音」：

1. 右键系统托盘 🔊 音量图标 → **声音设置** → **更多声音设置**。
2. 切换到 **录制** 选项卡 → 右键空白处 → **显示已禁用的设备**。
3. 找到 **立体声混音（Stereo Mix）** → 右键 → **启用**。
4. 在助手设备列表中选择它。
