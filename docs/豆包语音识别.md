# 豆包语音识别 (ASR)

除本地 **Whisper** 外，本助手支持使用 **火山引擎豆包流式语音识别（小时版）** 作为 STT 引擎。豆包 ASR 需在火山引擎控制台开通，并在 `backend/config.json` 中配置，**界面仅提供「识别引擎」切换（Whisper / 豆包），不展示密钥输入框**。

## 1. 开通与获取密钥

1. 打开 [火山引擎 - 豆包语音识别](https://www.volcengine.com/docs/6561/1354869) 文档，按指引开通「豆包流式语音识别模型 2.0 小时版」。
2. 在控制台创建应用，获取：
   - **App ID**（用于请求头 `X-Api-App-Key`）
   - **Access Token**（用于 `X-Api-Access-Key`）
3. 确认该应用已开通资源 **volc.seedasr.sauc.duration**（小时版）。

## 2. 配置文件

在 `backend/config.json` 中填写（仅当 `stt_provider` 为 `"doubao"` 时生效）：

```json
{
  "stt_provider": "doubao",
  "doubao_stt_app_id": "你的App ID",
  "doubao_stt_access_token": "你的Access Token",
  "doubao_stt_resource_id": "volc.seedasr.sauc.duration",
  "doubao_stt_boosting_table_id": ""
}
```

- `doubao_stt_resource_id`：固定为 `volc.seedasr.sauc.duration`（小时版）即可。
- `doubao_stt_boosting_table_id`：可选。若在控制台创建了热词表，将热词表 ID 填在此处，可提升专业术语识别率。

## 3. 热词表（可选）

项目提供 `backend/doubao_hotwords_面试技术词表.txt`，内含常见技术词（中英文）。上传为热词表后可减少专有名词识别错误。

**格式要求**：每行一个词，**不要权重、不要特殊符号**；仅字母/数字/中文，UTF-8 编码。

**上传步骤**：

1. 打开 [火山引擎 - 豆包语音 - 自学习平台 - 热词管理](https://console.volcengine.com/speech/hotword)。
2. 点击「添加热词文件」，输入名称（如：面试技术词表）。
3. 上传 `backend/doubao_hotwords_面试技术词表.txt`，或复制其内容粘贴。
4. 提交后，在列表中复制该词表对应的 **热词 ID**。
5. 在 `config.json` 中设置 `doubao_stt_boosting_table_id` 为上述 ID。

不填 `doubao_stt_boosting_table_id` 则不使用热词表，ASR 仍可正常使用。

## 4. 界面操作

在 **设置 → 语音识别引擎** 中选择 **「豆包（API）」** 即可切换。豆包相关密钥与热词 ID 均在 `config.json` 中配置，无需在界面填写。
