# Live2D Practice Model

Runtime: oh-my-live2d, MIT, https://github.com/oh-my-live2d/oh-my-live2d

Model: Natori from Live2D Cubism Web Samples, https://github.com/Live2D/CubismWebSamples/tree/develop/Samples/Resources/Natori

Live2D sample model use is governed by Live2D's sample model terms: https://www.live2d.com/en/learn/sample/model-terms/

The assets are vendored under `public/live2d/Natori` so the practice UI can render without runtime model CDN calls.
